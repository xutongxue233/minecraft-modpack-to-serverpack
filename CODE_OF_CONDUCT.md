# Code of Conduct

## Our Standards

We expect contributors to:

- be respectful and constructive;
- focus on technical facts and reproducible evidence;
- avoid harassment, insults, and personal attacks;
- respect maintainers' time and review decisions;
- avoid posting private keys, private server addresses, or other sensitive information.

## Unacceptable Behavior

Unacceptable behavior includes:

- abusive, discriminatory, or sexualized language;
- threats or harassment;
- repeated off-topic comments after maintainers ask to stop;
- publishing another person's private information;
- intentionally submitting malicious code, secrets, or misleading reports.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other contributions that violate this code. Serious or repeated violations may result in a temporary or permanent ban from the project.

## Reporting

Report conduct issues to the maintainers through GitHub. Security issues should be reported privately through GitHub Security Advisories, not public issues.
